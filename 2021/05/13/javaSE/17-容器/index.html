17-е®№еҷЁ

еҸ‘иЎЁдәҺ2021-05-13|жӣҙж–°дәҺ2021-08-16|java
|еӯ—ж•°жҖ»и®Ў:1.4k|йҳ…иҜ»ж—¶й•ҝ:5еҲҶй’ҹ|йҳ…иҜ»йҮҸ:

е®№ еҷЁ

CollectiionжҺҘеҸЈ

CollectionжҺҘеҸЈеЈ°жҳҺдәҶд»ҘдёӢж–№жі•пјҡ

жіЁпјҡListе’ҢSetжҳҜCollectionзҡ„еӯҗжҺҘеҸЈ

ListжҺҘеҸЈ

Listзү№зӮ№пјҡжңүеәҸгҖҒе…ғзҙ еҸҜйҮҚеӨҚ

  • жңүеәҸпјҡжҜҸдёӘе…ғзҙ йғҪжңүзҙўеј•ж Үи®°
  • ListйҖҡеёёе…Ғи®ёж»Ўи¶і e1.equals(e2) зҡ„е…ғзҙ йҮҚеӨҚеҠ е…Ҙе®№еҷЁ

зӣёжҜ”CollectionжҺҘеҸЈпјҢListдёӯеӨҡдәҶдёҖдәӣдёҺзҙўеј•жңүе…ізҡ„ж–№жі•

ArrayListзұ»

ArrayListеә•еұӮжҳҜз”ЁгҖҗж•°з»„гҖ‘е®һзҺ°зҡ„еӯҳеӮЁгҖӮ==зү№зӮ№пјҡжҹҘиҜўж•ҲзҺҮй«ҳпјҢеўһеҲ ж•ҲзҺҮдҪҺпјҢзәҝзЁӢдёҚе®үе…Ё==гҖӮ

жҲ‘д»¬зҹҘйҒ“пјҢж•°з»„й•ҝеәҰжҳҜжңүйҷҗзҡ„пјҢиҖҢArrayListжҳҜеҸҜд»Ҙеӯҳж”ҫд»»ж„Ҹж•°йҮҸзҡ„еҜ№иұЎпјҢй•ҝеәҰдёҚеҸ—йҷҗеҲ¶пјҢйӮЈд№Ҳе®ғжҳҜжҖҺд№Ҳе®һзҺ°зҡ„е‘ў?

жң¬иҙЁдёҠе°ұжҳҜйҖҡиҝҮе®ҡд№үж–°зҡ„жӣҙеӨ§зҡ„ж•°з»„пјҢе°Ҷж—§ж•°з»„дёӯзҡ„еҶ…е®№жӢ·иҙқеҲ°ж–°ж•°з»„пјҢжқҘе®һзҺ°жү©е®№гҖӮ

ArrayListзҡ„Objectж•°з»„гҖҗеҲқе§ӢеҢ–й•ҝеәҰдёә10гҖ‘пјҢеҰӮжһңжҲ‘д»¬еӯҳеӮЁж»ЎдәҶиҝҷдёӘж•°з»„пјҢйңҖиҰҒеӯҳеӮЁз¬¬11дёӘеҜ№иұЎпјҢе°ұдјҡе®ҡд№үж–°зҡ„й•ҝеәҰжӣҙеӨ§зҡ„ж•°з»„пјҢе№¶е°ҶеҺҹж•°з»„еҶ…е®№е’Ңж–°зҡ„е…ғзҙ дёҖиө·еҠ е…ҘеҲ°ж–°ж•°з»„дёӯ

LinkListзұ»

LinkListеә•еұӮз”ЁгҖҗеҸҢеҗ‘й“ҫиЎЁгҖ‘е®һзҺ°

==зү№зӮ№пјҡжҹҘиҜўж•ҲзҺҮдҪҺгҖҒеўһеҲ ж•ҲзҺҮй«ҳпјҢзәҝзЁӢдёҚе®үе…Ё==

еҸҢеҗ‘й“ҫиЎЁд№ҹеҸ«еҸҢй“ҫиЎЁпјҢжҳҜй“ҫиЎЁзҡ„дёҖз§ҚпјҢе®ғзҡ„жҜҸдёӘж•°жҚ®иҠӮзӮ№дёӯйғҪжңүдёӨдёӘжҢҮй’ҲпјҢеҲҶеҲ«жҢҮеҗ‘еүҚдёҖдёӘиҠӮзӮ№е’ҢеҗҺдёҖдёӘиҠӮзӮ№гҖӮ

жүҖд»ҘпјҢд»ҺеҸҢеҗ‘й“ҫиЎЁдёӯзҡ„д»»ж„ҸдёҖдёӘиҠӮзӮ№ејҖе§ӢпјҢйғҪеҸҜд»ҘеҫҲж–№дҫҝең°жүҫеҲ°жүҖжңүиҠӮзӮ№гҖӮ

Vectorзұ»

Vectorеә•еұӮжҳҜз”Ёж•°з»„е®һзҺ°зҡ„ListпјҢзӣёе…ізҡ„ж–№жі•йғҪеҠ дәҶеҗҢжӯҘжЈҖжҹҘпјҢеӣ жӯӨвҖңзәҝзЁӢе®үе…Ё,ж•ҲзҺҮдҪҺвҖқгҖӮ жҜ”еҰӮпјҢindexOfж–№жі•е°ұеўһеҠ дәҶsynchronizedеҗҢжӯҘж Үи®°гҖӮ

еҰӮдҪ•йҖүз”ЁArrayListгҖҒLinkedListгҖҒVector

  1. йңҖиҰҒзәҝзЁӢе®үе…Ёж—¶пјҢз”ЁVectorгҖӮ

  2. дёҚеӯҳеңЁзәҝзЁӢе®үе…Ёй—®йўҳж—¶пјҢе№¶дё”жҹҘжүҫиҫғеӨҡз”ЁArrayList(дёҖиҲ¬дҪҝз”Ёе®ғ)гҖӮ

  3. дёҚеӯҳеңЁзәҝзЁӢе®үе…Ёй—®йўҳж—¶пјҢеўһеҠ жҲ–еҲ йҷӨе…ғзҙ иҫғеӨҡз”ЁLinkedListгҖӮ

MapжҺҘеҸЈ

Mapе°ұжҳҜз”ЁжқҘеӯҳеӮЁвҖңй”®(key)-еҖј(value) еҜ№вҖқзҡ„гҖӮ Mapзұ»дёӯеӯҳеӮЁзҡ„вҖңй”®еҖјеҜ№вҖқйҖҡиҝҮй”®жқҘж ҮиҜҶпјҢжүҖд»ҘвҖңй”®вҖқдёҚиғҪйҮҚеӨҚгҖӮ

MapжҺҘеҸЈеёёз”Ёж–№жі•пјҡ

Map жҺҘеҸЈзҡ„е®һзҺ°зұ»жңүHashMapгҖҒTreeMapгҖҒHashTableгҖҒPropertiesзӯүгҖӮ

HashMapзұ»

HashMapйҮҮз”Ёе“ҲеёҢз®—жі•е®һзҺ°пјҢжҳҜMapжҺҘеҸЈжңҖеёёз”Ёзҡ„е®һзҺ°зұ»гҖӮ

з”ұдәҺеә•еұӮйҮҮз”ЁдәҶе“ҲеёҢиЎЁеӯҳеӮЁж•°жҚ®пјҢжҲ‘д»¬иҰҒжұӮй”®дёҚиғҪйҮҚеӨҚпјҢ==еҰӮжһңеҸ‘з”ҹйҮҚеӨҚпјҢж–°зҡ„й”®еҖјеҜ№дјҡжӣҝжҚўж—§зҡ„й”®еҖјеҜ№==

HashMapеңЁжҹҘжүҫгҖҒеҲ йҷӨгҖҒдҝ®ж”№ж–№йқўйғҪжңүйқһеёёй«ҳзҡ„ж•ҲзҺҮ

HashTableзұ»е’ҢHashMapз”Ёжі•еҮ д№ҺдёҖж ·пјҢеә•еұӮе®һзҺ°еҮ д№ҺдёҖж ·пјҢеҸӘдёҚиҝҮHashTableзҡ„ж–№жі•ж·»еҠ дәҶsynchronizedе…ій”®еӯ—зЎ®дҝқзәҝзЁӢеҗҢжӯҘжЈҖжҹҘпјҢж•ҲзҺҮиҫғдҪҺгҖӮ

HashMapдёҺHashTableзҡ„еҢәеҲ«

  1. HashMap: зәҝзЁӢдёҚе®үе…ЁпјҢж•ҲзҺҮй«ҳгҖӮе…Ғи®ёkeyжҲ–valueдёәnull

  2. HashTable: зәҝзЁӢе®үе…ЁпјҢж•ҲзҺҮдҪҺгҖӮдёҚе…Ғи®ёkeyжҲ–valueдёәnull

SetжҺҘеҸЈдёҺHashSetзұ»

SetжҺҘеҸЈз»§жүҝиҮӘCollectionпјҢSetжҺҘеҸЈдёӯжІЎжңүж–°еўһж–№жі•пјҢж–№жі•е’ҢCollectionдҝқжҢҒе®Ңе…ЁдёҖиҮҙгҖӮ

Setе®№еҷЁзү№зӮ№пјҡж— еәҸгҖҒдёҚеҸҜйҮҚеӨҚ

ж–°е…ғзҙ еҰӮжһңе’ҢSetдёӯжҹҗдёӘе…ғзҙ йҖҡиҝҮequals()ж–№жі•еҜ№жҜ”дёәtrueпјҢеҲҷдёҚиғҪеҠ е…Ҙ;

з”ҡиҮіпјҢSetдёӯд№ҹеҸӘиғҪж”ҫе…ҘдёҖдёӘnullе…ғзҙ пјҢдёҚиғҪеӨҡдёӘгҖӮ

Setеёёз”Ёзҡ„е®һзҺ°зұ»жңүпјҡHashSetгҖҒTreeSetзӯүпјҢжҲ‘д»¬дёҖиҲ¬дҪҝз”ЁHashSet

HashSetеҹәжң¬з”Ёжі•

иҝӯд»ЈеҷЁ

е°Ҷе®№еҷЁиҪ¬дёәиҝӯд»ЈеҷЁпјҡ

дҪҝз”ЁIteratorиҝӯд»ЈеҷЁйҒҚеҺҶе®№еҷЁе…ғзҙ (List/Set/Map)

  • иҝӯд»ЈеҷЁйҒҚеҺҶArrayList
  • иҝӯд»ЈеҷЁйҒҚеҺҶMap гҖҗ1гҖ‘
  • иҝӯд»ЈеҷЁйҒҚеҺҶMap гҖҗ2гҖ‘

Collectionsе·Ҙе…·зұ»

зұ» java.util.Collections жҸҗдҫӣдәҶеҜ№SetгҖҒListгҖҒMapиҝӣиЎҢжҺ’еәҸгҖҒеЎ«е……гҖҒжҹҘжүҫе…ғзҙ зҡ„иҫ…еҠ©ж–№жі•гҖӮ

  1. void sort(List) //еҜ№Listе®№еҷЁеҶ…зҡ„е…ғзҙ жҺ’еәҸпјҢжҺ’еәҸзҡ„и§„еҲҷжҳҜжҢүз…§еҚҮеәҸиҝӣиЎҢжҺ’еәҸгҖӮ
  2. void shuffle(List) //еҜ№Listе®№еҷЁеҶ…зҡ„е…ғзҙ иҝӣиЎҢйҡҸжңәжҺ’еҲ—гҖӮ
  3. void reverse(List) //еҜ№Listе®№еҷЁеҶ…зҡ„е…ғзҙ иҝӣиЎҢйҖҶз»ӯжҺ’еҲ— гҖӮ
  4. void fill(List, Object) //з”ЁдёҖдёӘзү№е®ҡзҡ„еҜ№иұЎйҮҚеҶҷж•ҙдёӘListе®№еҷЁгҖӮ
  5. int binarySearch(List, Object) //еҜ№дәҺйЎәеәҸзҡ„Listе®№еҷЁпјҢйҮҮз”ЁжҠҳеҚҠжҹҘжүҫзҡ„ж–№жі•жҹҘжүҫзү№е®ҡеҜ№иұЎгҖӮ
